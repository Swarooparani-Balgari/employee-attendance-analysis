# Employee Attendance Analysis

## Objective
The objective of this project is to analyze employee attendance data and identify attendance patterns and department performance.

## Tools Used
- Microsoft Excel
- Python (Pandas)
- Matplotlib

## Analysis Performed
- Data cleaning and preparation
- Attendance percentage calculation
- Department-wise comparison
- Employee performance analysis

## Key Insights
- Most employees attended more than 80% of working days
- Finance department has the highest attendance rate
- HR department has the lowest attendance rate
- One employee has the highest attendance percentage
